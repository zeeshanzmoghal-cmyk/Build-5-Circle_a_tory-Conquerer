# Project/App Name: Circle-a-tory Conqeurer

This is a smaller game designed to just pass time when the player if free of tasks. You, the shooter have to keep shooting the same falling circles to prevent them from hitting your "base". If they do hit the said "base" you lose. You cannot end a game once its started. to end, you have to lose first.

## How to Run the Game:
  - Clone  the repo
  - Create a new project
  - run main.py
